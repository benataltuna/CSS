Itxurak

Hurrengo adibidean CSS fitxategi ezberdina erabiliaz webgune baten itxura nola aldatu ikusiko dugu.

Beltza - Urdina - Horia

NOLA TXERTATU ESTILOA

1. Elementuan bertan

Style atributua erabiliz. Ez da egokia elementuak banaka-banaka etiketatu behar baitira. Zaila da webgunean estilo koherentea mantentzea eta aldaketak egin nahi baditugu kostu handia du.

							
								
Izenburua

2. HTML orrialdean bertan

CSS kodea HTML fitxategiko goiburuan (head) definitzen da, <style> </style> etiketen artean.

3. Kanpoko fitxategi batean

Kasu honetan CSSa, style.css moduko dokumentu batean definitzen da, HTML dokumentutik kanpo eta ondoren HTML dokumentuen burukoan lotura egiten da. Hau erabilita erraza da webgune osoan (HTML ezberdinak) itxura koherentea izatea bai eta aldaketak modu erraz eta erosoan egitea.

BACKGROUND

Background propietatea erabiliko dugu elementuen fondoa aldatzeko.

Propietate honekin orrialdearen fondoa, taula batena, paragrafo batena etab aldatu genitzake.

Kolorea aldatzeko


background-color: red;
background-color: #00ff00;

Irudi bat jartzeko


background-image: url("helbidea");
background-image: url("ona.jpg");

BACKGROUND

Erlazionatutako beste propietate batzuk:

  • background-position: Fondoko irudia non kokatu behar den zehazten du.
  • background-repeat: Fondoko irudia errepikatu behar den edo ez zehazten du.
  • background-attachment: Fondoko irudia finko utzi behar dugun zehazten du.

KUTXA EGITURA

HTML elementuak kutxa moduan ikus genitzake. Ondorengoa azpielementuek osatzen dute HTML elementu bat:

Elementuaren neurriak kalkulatzeko, azpielementu bakoitzaren neurriak batu behar dira.

MARGINAK

Elementu baten alboan jarri behar den espazioa (ertzetik kanpo)

					
						margin-top: 100px;
						margin-left: 5%;
						margin-right: inherit; /*Bere gurasoaren margin berdina*/

BORDER

Elementu ezberdinen ertzak aldatu daitezke. (4 berdin edo ezberdin)

					
						p {
							border-style: solid, dotted, double, etab;
							border-width: 5px;
							border-color: green;
							}
						p {
							border: solid 5px green;
						}
						p {
							border-left: dotted 3px blue;
						}

PADDINGA

Elementu baten eta bere ertzaren artean dagoen tartea.

					
						img {
							padding-top:100px;
							padding-left: 5%;
							padding-right: inherit; /*Bere gurasoaren padding berdina*/
						}

ERTZAK

Elementu ezberdinen ertzak aldatu daitezke. (4 berdin edo ezberdin)

					
						p {

ESTEKAK

Lau egoera ezberdinetan egon daiteke:

  • a:link: Normal, hau da, bisitatu edo klik egin gabe dagoenean.
  • a:visited: Jada bisitatu dugun esteka bat.
  • a:hover: Kurtsorea estekaren gainean dugunean.
  • a:active: Klikatua den momentuan.

a:hover beti a:link eta a:visited ondoren definitu behar da eta a:active beti a:hover ondoren.

ESTEKAK

  • background-color: Estekaren atzeko fondoaren kolorea aldatzeko erabiltzen da.
						
							a:hover{

								background-color: blue;
							}

a:hover beti a:link eta a:visited ondoren definitu behar da eta a:active beti a:hover ondoren.
